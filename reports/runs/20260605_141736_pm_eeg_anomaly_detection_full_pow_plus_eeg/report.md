# PM/EEG Anomaly Detection Report

## 1. Цель

Цель этого блока — закрыть задачу обнаружения паттернов или аномальных эпизодов в длинных EEG/PM-записях. Аномалии ищутся без ручной разметки, по резким изменениям PM-метрик и статистически необычным EEG/POW-признакам.

## 2. Метод

Для каждого окна рассчитываются несколько компонент anomaly score:

- robust z-score значения PM-метрики внутри записи;
- robust z-score абсолютного изменения PM-метрики между соседними окнами;
- отклонение PM-метрики от локального rolling mean;
- robust anomaly score по EEG/POW-признакам.

Итоговый `combined_anomaly_score` — это смесь PM anomaly score и feature anomaly score. Окна выше заданного квантильного порога помечаются как аномальные.

## 9. Интерпретация

Было оценено 43174 окон. Аномальными отмечено 864 окон (2.001%).

Такие окна можно интерпретировать как участки с резкой сменой состояния, нестабильной динамикой PM-метрик или необычным EEG/POW-профилем. Это не клиническая диагностика, а unsupervised способ выделения кандидатов на нестабильные или артефактные эпизоды для дальнейшего анализа.

## 10. Ограничения

- Метод не использует экспертную разметку аномалий.
- Аномальность определяется статистически, а не клинически.
- Feature anomaly score зависит от выбранного признакового пространства.
- Порог выбирается по квантилю и может быть изменён.
